## Задание 1. Yandex Cloud

1. Настроить с помощью Terraform кластер баз данных MySQL.

 - Используя настройки VPC из предыдущих домашних заданий, добавить дополнительно подсеть private в разных зонах, чтобы обеспечить отказоустойчивость. 
подсети по зонам locals.tf:
```
locals {
  project = "netology"
  env     = "hw15-4"
  mysql_cluster_name = "${local.project}-${local.env}-mysql"
  sg_mysql_name      = "${local.project}-${local.env}-mysql-sg"
  private_subnets = {
    a = { zone = "ru-central1-a", cidr = "10.10.20.0/24", name = "hw15-4-private-a" }
    b = { zone = "ru-central1-b", cidr = "10.10.21.0/24", name = "hw15-4-private-b" }
    d = { zone = "ru-central1-d", cidr = "10.10.22.0/24", name = "hw15-4-private-d" }
  }
  mysql_host_zones = ["a", "b"]
}
```
Зона ru-central1-c была DOWN, поэтому третья подсеть — в d, не в c.

 - Разместить ноды кластера MySQL в разных подсетях.
два хоста в подсетях a и b. mysql.tf:
```
resource "yandex_mdb_mysql_cluster" "main" {
  name        = local.mysql_cluster_name
  environment = "PRESTABLE"
  network_id  = yandex_vpc_network.main.id
  version     = var.mysql_version

  dynamic "host" {
    for_each = local.mysql_host_zones   # ["a", "b"]
    content {
      zone      = local.private_subnets[host.value].zone
      subnet_id = yandex_vpc_subnet.private[host.value].id
    }
  }
}
```

 - Использовать окружение Prestable, платформу Intel Broadwell с производительностью 50% CPU и размером диска 20 Гб.
variables.tf:
```
variable "mysql_resource_preset_id" {
  default     = "b1.medium"   # Intel Broadwell, 2 vCPU, 50% CPU, 4 GB RAM
}

variable "mysql_disk_size" {
  default = 20
}
```

mysql.tf:
```
  environment = "PRESTABLE"

  resources {
    resource_preset_id = var.mysql_resource_preset_id
    disk_type_id       = var.mysql_disk_type_id   # network-ssd
    disk_size          = var.mysql_disk_size      # 20
  }
```

 - Задать время начала резервного копирования — 23:59.
mysql.tf:
```
  backup_window_start {
    hours   = 23
    minutes = 59
  }
```

 - Включить защиту кластера от непреднамеренного удаления.
mysql.tf:
```
  deletion_protection = true
```

 - Создать БД с именем `netology_db`, логином и паролем.
mysql.tf:
```
resource "yandex_mdb_mysql_database" "netology" {
  cluster_id = yandex_mdb_mysql_cluster.main.id
  name       = var.mysql_db_name    # netology_db
}

resource "yandex_mdb_mysql_user" "netology" {
  cluster_id = yandex_mdb_mysql_cluster.main.id
  name       = var.mysql_user       # netology
  password   = var.mysql_password

  permission {
    database_name = yandex_mdb_mysql_database.netology.name
    roles         = ["ALL"]
  }
}
```
